# Play Console Field Checklist

Use this when filling the first Play Console draft.

## Main store listing

- App name
- Short description
- Full description
- App icon
- Feature graphic
- Phone screenshots
- Contact email
- Website
- Privacy policy URL
  - `https://ngcat.uk/privacy-policy.html`

## App content

- Privacy policy
- Ads declaration
- Content rating questionnaire
- Target audience and content
- News declaration: likely not applicable

## App access

- Explain how reviewers can reach the main functionality
- Mention that the app requires Android Notification Access
- Mention that forwarding channels are optional and can be tested from settings

## Data safety

- Use `docs/GOOGLE_PLAY_DATA_SAFETY_NOTES.md`
- Make sure answers match:
  - in-app disclosure
  - public privacy policy
  - actual runtime behavior

## Testing tracks

- Internal testing first
- Add tester emails
- Verify:
  - onboarding disclosure
  - notification access flow
  - one successful channel test send
  - one real notification forwarding case
